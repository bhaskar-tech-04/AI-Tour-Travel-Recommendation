# 📌 About Project

AI Tour & Travel Recommendation System is a smart web application developed using Flask and APIs that helps users explore travel destinations easily.

Users can enter any destination name and instantly get:

* Beautiful destination image
* Travel package details
* Hotel information
* Budget details
* Activities
* Best visiting time

The project provides an easy and interactive travel planning experience.

---



# 🚀 Features

✅ Search travel destinations

✅ Dynamic destination images using Unsplash API

✅ Travel package recommendations

✅ Budget information

✅ Hotel suggestions

✅ Best time to visit

✅ Activities recommendations

✅ Responsive modern UI
